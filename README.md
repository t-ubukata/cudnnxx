# cudnnxx

cuDNN C++ wrapper.

## Features

- Header only.
- Only includes host codes.
- Written in C++11.
- Mainly focused on RAII.

## Requirements

- x86_64 Linux. (Tested on Ubuntu 16.04 LTS)
- gcc or Clang (Tested on Clang 3.8.0)
- CUDA Toolkit 9.1 or later.
- cuDNN 7.1.2 or later.

## Build and test

```
make
```
